# AI-Powered Job Market & Skill Demand Analyzer

## 📌 Project Overview

The AI-Powered Job Market & Skill Demand Analyzer is a Data Analytics project focused on analyzing real-world job market datasets to uncover hiring trends, salary patterns, in-demand skills, and opportunities for aspiring data analysts.

This project aims to simulate a real-world business intelligence workflow using data cleaning, exploratory data analysis, and interactive dashboard development.

---

# 🎯 Objectives

The main objectives of this project are:

* Analyze job market trends for data-related roles
* Identify the most in-demand technical skills
* Explore salary trends across locations and industries
* Discover top hiring companies and locations
* Understand entry-level opportunities for freshers
* Build interactive dashboards using Power BI
